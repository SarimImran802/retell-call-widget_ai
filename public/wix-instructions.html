Wix Integration Instructions

Important: For microphone access to work properly in Wix, you need to configure the iframe with specific permissions.

Method 1: Custom HTML Element (Recommended)

In your Wix site:

  1. Add a "Custom HTML" element to your page
  2. Use this code:
<iframe src="https://retell-call-widget-ai.vercel.app/home.html" width="100%" height="600px" frameborder="0" allow="microphone; camera" allowfullscreen> </iframe>

Method 2: HTML Embed Code

If using HTML embed:

<iframe src="https://retell-call-widget-ai.vercel.app/home.html" width="100%" height="600px" frameborder="0" allow="microphone; camera" sandbox="allow-same-origin allow-scripts allow-forms allow-popups allow-presentation" allowfullscreen> </iframe>

Key Points:

  • allow="microphone; camera" - This is crucial for microphone access
  • sandbox attributes - Allow necessary permissions for the iframe
  • HTTPS required - Both your Wix site and the widget must use HTTPS
Testing: After implementing, test the microphone access by:
  1. Clicking "Start Call"
  2. Allowing microphone access when prompted
  3. Speaking to test if the AI responds

Troubleshooting:

  • No microphone prompt: Check iframe permissions and HTTPS
  • Permission denied: User needs to manually allow microphone access
  • AI not responding: Check browser console for errors
